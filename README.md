#  SQL Sales Analytics Project

##  Project Overview
This project focuses on analyzing sales data using **SQL** to extract meaningful business insights such as revenue trends, top-selling products, customer behavior, and order performance.

##  Tools Used
- MySQL
- SQL (Joins, Aggregations, Subqueries)
- GitHub

##  Dataset
The dataset contains information about:
- Orders
- Order details
- Products / Pizzas
- Pricing

##  Key Analysis Performed
- Total revenue calculation
- Top-selling products by quantity and revenue
- Daily and monthly sales trends
- Average order value
- Category-wise sales contribution
